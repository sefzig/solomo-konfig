Solomo Konfig
=============

Erst drucken, dann überlegen: Konfigurierbare QR-Aufkleber. 
Weiterleiten, eigener Text, Iframe, Twitter, Youtube, Maps. 
Erzeugt ein Wurmloch zum digitalen Paralleluniversum.
